# Progress Point – Task Manager

**Progress Point** is a simple task manager that helps you organize your day, prioritize tasks, and track progress. It encourages discipline, balance, and consistent productivity through visual stats and easy task management.

---

## Features

* Add tasks with title, description, date, and priority
* Mark tasks as High, Medium, or Low priority
* Track completed vs. pending tasks and daily streaks
* Filter tasks by priority, status, or date
* Replicate recurring tasks quickly
* Light & dark theme toggle
* Responsive design for mobile & desktop
* Motivational quotes to keep you going

---

## How It Works

1. Add tasks with relevant details.
2. Track your daily progress through stats and streaks.
3. Mark tasks as done, edit, delete, or replicate them.
4. Filter tasks to focus on what matters most.

---

## Tech Stack

* **HTML5 & CSS3** – Structure and styling
* **JavaScript** – Dynamic task management
* **LocalStorage** – Save tasks and theme preferences

---

## Getting Started

1. Clone the repo:

   ```bash
   git clone <repository-url>
   ```
2. Open `index.html` in your browser.
3. Start adding and managing your tasks.
